# 参数取值

## --category（类别）

| 值        | 说明                  |
| --------- | --------------------- |
| `emacs`   | Emacs 相关            |
| `python`  | Python 相关           |
| `rust`    | Rust 相关             |
| `go`      | Go 相关               |
| `android` | Android 开发相关      |
| `scheme`  | Guile/Scheme 相关     |
| `shell`   | Shell 脚本            |
| `nix`     | Nix/NixOS 相关        |
| `guix`    | Guix System 相关      |
| `web`     | 前端/Web              |
| `general` | 通用/跨领域           |

## --tech（技术栈）

`emacs-lisp` | `python` | `rust` | `go` | `kotlin` | `java` | `gradle` | `scheme` | `guile` | `bash` | `nix` | `typescript` | `org-mode`

## --type（类型）

| 值         | 说明        |
| ---------- | ----------- |
| `debug`    | 调试/排障   |
| `refactor` | 重构        |
| `research` | 调研/探索   |
| `workflow` | 工作流/流程 |
| `feature`  | 新功能开发  |
| `config`   | 配置调整    |

## --owner（执行者）

| 值              | 说明                |
| --------------- | ------------------- |
| `ai`            | AI 独立完成（默认） |
| `human`         | 人工独立完成        |
| `collaborative` | 人机协作            |
